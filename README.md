# To-Do Notebook

## Author
Jonathan Ortiz Guzman

## Description
An app where one can write stuff down daily and long term to keep track of what needs to be done that day and for what categories such as classes or home. This is the digital implementation of my notebook where I write down what I need to do on a certain day, the category and the time block I spend for the task.

### Features
- Has two tabs where it shows long term tasks and current day tasks
- Has the ability to cross out tasks when clicked on
- Have the ability to go back to previous days to see those day's tasks
- Ability to add/edit/remove tasks (Task Name, Category, Time Spent)
- When opening, shows the current day
- Store data locally
